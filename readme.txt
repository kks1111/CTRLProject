# Web-Development Project Control Expense Distribution

## SETUP

1. First install XAMPP Apache Distribution from Internet.			
2. Start the Apache and MySQL modules using the XAMPP controller.
3. Open the browser and type "localhost/phpmyadmin" and it opens the "phpmyadmin" page.Then create a database "ctrlproject". 
4. Import the "ctrlproject.sql" file present in the zip folder.
5. Open the "htdocs" folder in the "xampp" folder. Copy paste the folder "webproject" in "htdocs" folder.
6. Open the browser and type "localhost/webproject/php/indexpage.php" and you should see the index page of the website.
